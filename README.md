## Мой первый сайт с нейросетью - сетчик кино по настроению пользователя

### Использоание
Для использования моего сайта необходимо скзать архив с файлами, сгенерировать системные файлы `.idea` и `папка venv`
для установки всех библиотек необходимо ввести команду 
```
pip install -r requirements.txt
```
